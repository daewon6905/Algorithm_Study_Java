난이도 : lvl3

다익스트라 알고리즘 변형. 기존엔 특정 노드까지의 누적된 값을 최소/최대화하는 방향이였다면, 
이 문제는 목표 노드(산봉우리)까지의 노드와 노드 사이의 거리의 최댓값을 갱신해주면 됨. 산봉우리까지 올라가는 경우만 봐도 무방한 이유는, 어짜피 같은 경로로 내려오거나 같은 산봉우리로 올라가는 다른 경로가 있더라도 그 경우도 큐에 들어가서 값이 갱신된 상태이기 때문에, 내려오는 케이스는 고려하지 않아도 됨. 또한, 시작점을 큐에 모두 넣고 시작하기에, 다른 출발점에 의한 경우도 반영됨. 

**문제의 조건중에 Intensity가 같다면, 작은 수의 산봉우리가 정답이라는 조건 때문에 마지막 Intensity의 최소를 구할때 조건식에 조건을 추가하거나 summits 배열을 정렬을 한 뒤(오름차순) 답을 도출 해야함.**
