**문제 유형**: 자료구조 (Stack)

시간 복잡도: $O(N)$

**📌 문제 설명**

주어진 문자열 s는 '(', ')', '{', '}', '[', ']'로만 이루어져 있습니다. 이때, 문자열 s에서 올바른 괄호 쌍의 개수를 반환하는 solution 함수를 작성하세요. 만약 문자열 s가 올바르지 않은 괄호로 구성되어 있다면 -1을 반환합니다.

⚠️ 올바른 괄호의 정의여는 괄호 '(', '{', '['는 각각 닫는 괄호 ')', '}', ']'와 반드시 짝지어야 합니다.여는 괄호는 올바른 순서로 닫혀야 합니다. (예: ({[]}) → 올바름, )( → 올바르지 않음)

💡 **풀이 전략** 

보통 스택 문제에서는 여는 괄호를 스택에 넣고 닫는 괄호를 만났을 때 조건문(if-else)으로 분기하여 스택의 탑과 비교합니다. 하지만 이 풀이에서는 코드를 훨씬 더 간결하게 만들기 위해 아래와 같은 방식을 사용합니다.

✨ 스택 역발상 매칭 기법
여는 괄호를 만났을 때: 스택에 여는 괄호를 그대로 넣는 대신, 나중에 나와야 할 닫는 괄호 짝을 미리 스택에 push합니다.

( 가 오면 ➔ ) 를 push

{ 가 오면 ➔ } 를 push

[ 가 오면 ➔ ] 를 push

닫는 괄호를 만났을 때: 현재 문자(c)가 스택의 탑(stack.peek())과 일치하는지 바로 확인합니다.

!stack.isEmpty() && stack.peek() == c 가 성립하면 올바른 짝을 찾은 것이므로, 스택에서 제거(pop)하고 괄호 쌍의 개수(count)를 1 증가시킵니다.

만약 스택이 비어있거나 짝이 맞지 않는다면 곧바로 올바르지 않은 문자열이 되므로 -1을 반환합니다.

최종 검사: 문자열 순회가 끝난 후 스택이 완전히 비어있다면(stack.isEmpty()) 성공적으로 모든 짝을 맞춘 것이므로 count를 반환하고, 남아있는 게 있다면 -1을 반환합니다.
