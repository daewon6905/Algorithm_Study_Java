난이도: Hard

노드와 노드 사이의 노드의 개수를 간선의 가중치로 보고 최대한 많은 노드를 방문하려면 최단거리로 이동해야하므로 다익스트라 알고리즘으로 각 노드까지의 최단거리를 계산한다.
여기까지는 생각을 했는데 그 후 방문 노드 카운팅이 까다롭다. 방향이 없는 그래프 이므로 한쪽에서 오는 경우와 그 반대 쪽에서 오는 경우를 고려하여 만약 양쪽에서 들어와서 그 사이의 노드 개수를 넘었다면, 사이에 있는 노드 개수를 더하고, 만약 양쪽에서 들어와서 못만난다면 양쪽 각각 값을 더함.

이걸 결과값에 **answer += Math.min(d, remainU + remainV);** 최솟값을 결과에 더해준다.
