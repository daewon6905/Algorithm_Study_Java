난이도 : Hard

어려움. 우선순위 큐와 슬라이딩 윈도우를 사용.

모든 리스트에서 적어도 하나를 포함해야 하는점이 까다로움. 우선순위 큐에 이미 오름차순으로 정렬되어있는 리스트의 첫 요소를 넣어 최솟값을 구하고, 현재 최댓값은 따로 구한다.큐에서 하나씩 꺼낼때, 현재 구간이 최소인지 확인 후, range 배열과 minRange 값을 업데이트한다.
그 후, 현재 최댓값과 큐에서 꺼낸 요소가 속한 리스트의 다음 요소를 비교하여 최댓값을 업데이트 후 큐에 다음 요소를 넣어줌. 종료 조건은 하나의 리스트에서라도 더 꺼낼 숫자가 없으면 종료함. 이유는 모든 리스트의 값을 포함할 수 없기 때문
