난이도 : lvl2 ~ lvl3

전형적인 bfs를 사용하는 최단거리 문제 같지만, 2칸을 넘을 수 있는 경우가 존재하므로, 그 상태를 2차원 visited 배열로는 부족하다. **3차원을 고려해야 해결**이 가능하다.

**boolean[][][] visited = new boolean[n][m][2]** 와 같이 기존 2차원에 2칸 넘기를 사용 유무에 따른 2가지 경우를 나타내는 3차원 visited 배열을 사용한다.

상태를 포함한 BFS(상태 공간 탐색) 문제. 2칸을 건너뛰는 아이템 사용 유무를 3차원 배열로 생각해낼 수 있어야한다는 점이 어려운 포인트인듯
